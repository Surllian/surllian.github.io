Overview:

This smart garden is using Arduino to calculate the humidity and temperature of the garden. After 1 minutes, the Arduino will push data to Thunkable (an IOS creating app) and it will illustrate the garden’s humidity and temperature. Moreover, the clock has 5 led lights that light on by pressing button in Thunkable app, or it will automatically turn on when the light sensor detects no light in the garden (as same as the garden light). Like normal soil calculator, the smart garden can calculate soil humidity and send notification if the soil humidity is low or over the range of permission. The user can active the automatically watering function by pressing button on Thunkable apps.

Motivation:

Firstly, this smart garden project’s budget is lower (around ~500k) than another watering system, which use in agriculture industry. Moreover, the area to set up this smart garden is small, so it will be more efficient for people, who love to create a mini garden but cannot afford a big scale garden. In addition, researching and building a smart garden by yourself is very interesting, as a developer, I will gain more experience and knowledge about Arduino aspect, which helps me to discover more about the world of IoT and robotic.

IoT – Internet of Things:
According to BBC , the 'Internet of Things' is the concept of networking lots of devices so that they can collect and transmit data. The idea that any object or living being can be uniquely identified on the internet is central to the concept. By automating the capture of information, greater quantities of it can be stored and processed. Internet of Things can contain monitoring sensors, people or object that is tracked by RFID tag, or store some database contents. In this project, developers will create an online buffer storage to store the information that collected from Arduino. Then, the information will convert to an application and this app will illustrate the data.

Working with ThingSpeak:
 According to ThingSpeak, ThingSpeak is an IoT analytics platform service that allows people to aggregate, visualize, and analyze live data streams in the cloud. Users can send data to ThingSpeak from their devices, create instant visualization of live data, and send alerts. ThingSpeak will collect data and send sensor data privately to the cloud. Moreover, this website will analyze and visualize the data with MATLAB and trigger then reaction. To create a buffer storage, developer has to register for the ThingSpeak account. After that, create a new channel to store the information. In the new channel section, user can create from 1 to 8 fields, each fields will store different data. After filling the name of each field, the user need to add the metadata, the information of channel data, it can be JSON, XML, or CSV data.

Working with Thunkable:
 On Thunkable, user can develop an IOS or Android app by using drag and drop block. After finish the project, developer can publish the application on CH Play or IOS by if the application size is not exceeded 50MB, also people can store application on Thunkable at the limit 200MB for free. This project will use Thunkable for creating an IOS application which can extract data from ThingSpeak to developer’s app.
