Basic Git Commands: Setup & Crucial

Setup:

1. See where is git located: which git
2. View Version of Git: --version
3. Create alias for Git status: Git config
4. Support Centre: git help

Crucial:

1. Set everything ready to commit: git add .
2. Commit changes: git commit -m "description"
3. Update changes: git add -u
4. Move or rename: git mv oldname.php dir/newname.php
5. Receive newest data update: git pull
6. Clone a reposity to Visual Code Studio: git clone "URL link"
Source!

Basic Git Commands: Reset & Update

Reset:

1. Undo previous commit: git reset --soft HEAD~
...

Update:

1. Commit and update to recent commit: git commit --amend -m "description"
2. Upload: git commit -m "description"

Branching

Branching:

1. List all the branches: git branch
2. Rename branch1: -m OLDNAME NEWNAME
3. Rename branch2: git branch -- move OLDNAME NEWNAME
4. New branch: git branch NEWBRANCHNAME
5. Change branch: git checkout BRANCHNAMETHATYOUWORK
6. Delete branch: git branch -D BRANCHNAME
7. Delete merged branch1: git branch -d NAME
8. Delete merged branch2: git branch --delete NAME
9. Merge branch: git merge NAME
Source!

Conflict Resolve

Conflict Resolve:

Same Name File:

1. Do as usual : git add -> git commit -> git push
2. If there was error, add this command: git pull
3. Redo step 1 (or you can the file name ^^)

Merge Conflict:

1. Get branch script and name it "Bla":
git checkout feature_X -- V42__migration.sql mv Bla__migration.sql Bla__migration.sql
2. Get "Bla" script from previous merged commit : git checkout HEAD -- Bla__migration.sql
3. Ensure the index has updated sucessfully: git -> git commit
